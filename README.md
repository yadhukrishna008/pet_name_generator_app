Pet Name Generator (LangChain + Ollama + Streamlit)

A simple AI-powered Pet Name Generator built using LangChain, Ollama (local LLM), and Streamlit.  
The app generates creative pet names based on the selected animal type and color using a structured prompt and modern LangChain Runnable chains.

---

Features:
- Uses **local LLMs via Ollama** (no OpenAI API key required)
- Built with **modern LangChain (Runnable API)**
- Simple and interactive **Streamlit UI**
- Supports multiple pet types and custom colors
- Controlled output format using prompt engineering

---

Tech Stack
- **Python 3.10+**
- **LangChain Core**
- **LangChain Ollama**
- **Ollama (LLM runtime)**
- **Streamlit**
- **python-dotenv**

---

Prerequisites
- Python 3.10 or higher
- Download from: https://ollama.com
- Required LLM model pulled : ollama pull llama3
